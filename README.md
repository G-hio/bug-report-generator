Herramienta diseñada para estandarizar y agilizar la creación de reportes de errores durante el ciclo de pruebas de software. 

###  Funcionalidades QA
* **Estandarización:** Reportes con severidad, prioridad y pasos de reproducción.
* **Validación:** Lógica integrada para asegurar datos críticos del bug.
* **Exportación:** Formatos listos para herramientas de seguimiento.
